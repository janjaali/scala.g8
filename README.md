# scala.g8

[Giter8] template skeleton for scala projects.

## Usage

Giter the template

```shell
g8 janjaali/scala.g8
```

## Test template locally

To test the template locally create a new directory and run:

```shell
mkdir testDir
cd testDir
g8 file://../scala.g8/
```

[Giter8]: http://www.foundweekends.org/giter8/
